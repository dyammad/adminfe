# Sistema de Gestão da Igreja

Um sistema completo para gerenciamento de igrejas com interface moderna e funcionalidades abrangentes.

## 🚀 Funcionalidades

### 📊 Dashboard
- Visão geral com estatísticas importantes
- Gráficos de crescimento de membros
- Relatórios financeiros mensais
- Métricas de células e ministérios

### 👥 Gestão de Membros
- **Membros Ativos**: Lista completa com informações detalhadas
- **Membros Inativos**: Controle de membros que se afastaram
- **Visitantes**: Registro de pessoas que visitaram a igreja
- Funcionalidades de conversão de visitantes em membros

### 🏛️ Organização
- **Ministérios**: Gestão completa dos ministérios da igreja
- **Líderes**: Cadastro e controle da liderança
- **Células**: Administração dos grupos de células
- Vinculação de membros às células

### 📅 Eventos e Agenda
- **Eventos**: Criação e gestão de eventos da igreja
- **Agenda do Pastor**: Controle da agenda pastoral
- Calendário integrado com compromissos

### 💰 Gestão Financeira
- **Tesouraria**: Controle completo de receitas e despesas
- **Doações**: Registro de dízimos, ofertas e doações especiais
- Relatórios financeiros detalhados
- Categorização de transações

### 🙏 Ministério Pastoral
- **Pedidos de Oração**: Sistema para gerenciar pedidos de oração
- **Batismos**: Controle de batismos agendados e realizados
- Status de acompanhamento

## 🚀 Como Usar

1. **Abra o arquivo `index.html` em um navegador web**
2. **Faça login com as credenciais padrão**
3. **Navegue pelas seções usando o menu lateral**
4. **Adicione dados usando os botões "Adicionar"**
5. **Visualize relatórios no Dashboard**

## 🐉 Ifrit Inventory — Coleção Final Fantasy

O repositório inclui uma mini‑aplicação estática para organizar coleções de Final Fantasy.

• Caminho: `ifrit-inventory/`

• Como abrir:
  1. Abra `ifrit-inventory/index.html` no navegador
  2. Use a barra de busca, filtros e categorias (Jogos, Livros, Loterias FFVII Remake/FFXVI, etc.)
  3. Ordene por nome, ano ou raridade

• Observações de deploy (Hostinger):
  - Envie a pasta `ifrit-inventory/` para `public_html/`
  - Acesse por `https://seu-dominio/ifrit-inventory/`
  - Garanta que permissões estejam em 755 (pastas) e 644 (arquivos)
